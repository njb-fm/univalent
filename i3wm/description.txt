Channel Name : i3wm

This is a flavour which installed i3-gaps as a  window manager.
i3-gaps is a modification of the original i3-wm and includes functions such as gaps between windows.
Also, this edition don't include LibreOffice, Rhythmbox, and so on. as standard because this is focused on running on netbook. 

デスクトップ環境…というかウインドウマネージャーにi3-gapsを採用したエディションです。
i3-gapsは本家i3-wmを改造したモノで、例えばウインドウの間を空けるなどといった機能が搭載されています。
またこのエディションはネットブックなどでの実行に重きをおいてある故、LibreOfficeやRhythmboxなどは搭載してありません。
